# Movie Streaming Hub

## Features
- User Registration
- Search Movies
- Watch Trailer
- Rate Movies
- Favorites List
- User Reviews
- Responsive Design

## Installation
1. **Clone the Repo**: `git clone https://github.com/{owner}/Movie-Streaming-Hub.git`
2. **Install Dependencies**: Run `npm install` in the project directory.
3. **Start the Server**: Use `npm start` to launch the application.

## Usage
- Navigate to `http://localhost:3000` in your browser.
- Register a new account or log in.
- Explore various movies and add them to your favorites.
